
# Exercícios do 1º Semestre de Sistemas de Informação

Este repositório contém os exercícios práticos realizados durante o 1º semestre do curso de Sistemas de Informação. Os exercícios estão organizados em diferentes pastas, cada uma dedicada a um roteiro prático específico, cobrindo os principais tópicos abordados ao longo do semestre.

## Estrutura das Branches

- **while**: Roteiro prático sobre estruturas de repetição `while`. 
- **condicionais**: Roteiro prático sobre estruturas condicionais (`if`, `else`, `elif`).
- **listas-1**: Roteiro prático sobre listas (parte 1).
- **listas-2**: Roteiro prático sobre listas (parte 2).
- **funcoes**: Roteiro prático sobre funções e modularização.
- **matrizes**: Roteiro prático sobre o uso de matrizes.
- **revisao**: Roteiro prático de revisão, abordando os tópicos vistos até o momento.
- **manipulacao-string**: Roteiro prático sobre manipulação de strings.

## Como acessar as branches

Para acessar os exercícios de cada tópico, siga os passos abaixo:

1. Clone o repositório:
    ```bash
    git clone https://github.com/seu-usuario/nome-do-repositorio.git
    ```
   
2. Acesse a branch desejada:
    ```bash
    git checkout nome-da-branch
    ```

Por exemplo, para acessar os exercícios de condicionais, utilize o comando:
```bash
git checkout condicionais
```

## Pré-requisitos

Certifique-se de ter instalado:

- Python 3.x
- Um editor de texto ou IDE, como o Visual Studio Code.

## Como executar os exercícios

1. Navegue até o diretório onde clonou o repositório:

    ```bash
    cd nome-do-repositorio
    ```
2. Acesse a branch desejada, conforme instruído acima.
3. Execute os scripts Python diretamente no terminal ou usando a IDE de sua escolha.
